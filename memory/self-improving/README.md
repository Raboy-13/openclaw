# Self-Improving Memory System

## 3 Layers

| Layer | File | Purpose | Load |
|---|---|---|---|
| Hot | `memory/self-improving/hot.md` | Important confirmed rules | Always |
| Context | `memory/self-improving/context.md` | Project/domain rules | Per topic |
| Archive | `memory/self-improving/archive.md` | Inactive patterns | Never (backup) |

## Learning Log
- File: `memory/self-improving/learning-log.md`
- Format: `[timestamp] | rule summary | context | source`

## Rules

1. **Learn only from explicit corrections/preferences** — never assume from silence
2. **Log every correction** with timestamp, context, source location
3. **Track repetition** — if same correction appears 3×, ask about permanent rule
4. **Weekly maintenance** — dedupe, compact confirmed rules, archive stale items, send digest
5. **Cite source on apply** — always mention file + line when using a learned rule
6. **No sensitive data** — never store passwords, tokens, financial, health, or private info
7. **Forget on request** — "forget X" → delete everywhere, confirm. "forget everything" → wipe and restart
8. **High-impact changes** — ask before applying
